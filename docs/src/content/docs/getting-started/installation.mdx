import { Tabs, TabItem } from '@astrojs/starlight/components';

Install Solid Stripe.js and the Stripe.js loader from the [npm public registry](https://www.npmjs.com/package/solid-stripe).

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @stripe/stripe-js solid-stripe
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @stripe/stripe-js solid-stripe
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @stripe/stripe-js solid-stripe
    ```
  </TabItem>
</Tabs>

In your server-side app, add the official server-side/node version of Stripe too:

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install stripe
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add stripe
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add stripe
    ```
  </TabItem>
</Tabs>

We'll use [solid-start](https://start.solidjs.com) in all of the examples.
